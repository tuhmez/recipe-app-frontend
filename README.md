# Recipe App Frontend

## Usage

Install:

```bash
npm i
```

### Start

```bash
npm run start
```

### Build

```bash
npm run build
```

### Production Start

```bash
npm run start:prod
```

### Storybooks

Build Storybook:

```bash
npm run build-storybook
```

Run Storybook:

```bash
npm run storybook
```
